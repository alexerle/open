ZHZ-Dashboard-API

Zentrale API fuer die ZHZ-Portlet-Suite. Diese API stellt Daten fuer das Plugin-Dashboard bereit.

Verfuegbare Endpunkte:

Alle Daten /data/dashboard.json
Portlets /data/portlets.json
Ankuendigungen /data/announcements.json
Unternehmen /data/company.json
Dokumentation /data/documentation.json
